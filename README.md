#  Password Strength Tester

A Python-based password strength analysis tool built using Dropbox's `zxcvbn` library.  
It evaluates a passwords entropy, pattern complexity, and estimated crack time to help users understand how secure their passwords really are.

##  Features
- Detects weak/common password patterns  
- Estimates brute-force crack time  
- Suggests improvements  
- Visualizes password score

## О Tech Stack
- Python 3
- Google Colab
- zxcvbn
